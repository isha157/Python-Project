# 📱 WhatsApp Message Automation (Python)

This project automates sending WhatsApp messages using Python and the Twilio API.

---

## 🚀 Features
- Send automated WhatsApp messages
- Uses Twilio API for messaging
- Accepts recipient details from user input
- Simple Python automation script

---

## 🛠 Technologies Used
- Python
- Twilio API
- Datetime module

---

## 📂 Project Structure

Python-Project
│
├── main.py      # Main script to send WhatsApp messages
├── config.py    # Configuration file
└── README.md    # Project documentation

---

## ⚙️ Installation

1. Clone the repository

git clone https://github.com/isha157/Python-Project.git

2. Install required library

pip install twilio

3. Run the program

python main.py

---

## 🔐 Important

Do NOT upload your Twilio credentials publicly.

Replace credentials locally like this:

account_sid = "YOUR_TWILIO_ACCOUNT_SID"
auth_token = "YOUR_TWILIO_AUTH_TOKEN"

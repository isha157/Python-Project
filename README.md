# WhatsApp Message Automation using Python

This project sends automated WhatsApp messages using the Twilio API and Python.

## Features
- Send WhatsApp messages automatically
- Uses Twilio API for messaging
- Takes recipient details from user input
- Simple Python script for automation

## Technologies Used
- Python
- Twilio API
- Datetime module

## How It Works
1. The user enters the recipient name and phone number.
2. The program connects to Twilio using API credentials.
3. The message is sent automatically through WhatsApp.

## How to Run
1. Install the required library:
   pip install twilio

2. Add your Twilio Account SID and Auth Token.

3. Run the Python script:
   python main.py

## Project Structure
main.py – main program  
config.py – configuration file
